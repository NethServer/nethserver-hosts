Cartelle condivise

Crea, modifica o elimina cartelle condivise (o i-bay).

Nome cartella condivisa
Il nome cartella condivisapuò contenere solo lettere minuscole, numeri, punti, trattini e underscore e deve iniziare con una lettera minuscola. Per esempio "brambilla", "intra" e "prog.soft" sono tutti nomi validi, mentre "2dipicche", "Mario Rossi" e "go!web" non lo sono. Il nome deve essere limitato a 12 caratteri.
Cestino di rete
Quand abilitato, i file eliminati dalla cartella condivisa vengono spostati in una directory apposita che funziona da cestino. E' quindi possibile ripristinare i file erroneamente cancellati dal cestino.
Cestino - Non sovrascrivere file esistenti
Quando un file viene eliminato, se nel cestino esiste già un file con lo stesso nome, tale file rinominato garantendo il ripristino di entrambi i file.
Shadow copy
Le Shadow Copy (dette anche Versioni Precedenti) sono una funzionalità, introdotta in Windows XP SP1 e versioni successive, che permette di effettuare dei backup automatici di file e cartelle in vari momenti temporali. E' così possibile tenere traccia di varie versioni di uno o più file. Il sistema provvede a creare backup automatici della cartella condivisa ogni ora.
Esegui contenuti dinamici
Consente l'esecuzione di script e CGI all'interno delle cartelle condivise. E' utile qualora si vogliano ospitare, ad esempio, siti in PHP all'interno delle cartelle condivise.
Permessi di lettura
Indica a quali utenti o gruppi hanno il permesso di lettura sulla cartella condivisa:
  • Amministratore: solo l'amministratore (utente admin)
  • Gruppo: solo il gruppo proprietario della cartella condivisa
  • Tutti
Permessi di scrittura
Vale quanto descritto nei 'Permissi di lettura'.
Accesso

Le cartelle condivise possono essere configurare per garantire due tipi di accesso:

  • Accesso da rete locale
    • Richiedi password
    • Richiedi password solo per reti remote
    • Non richiedere la password
  • Accesso da qualsiasi rete
    • Richiedi password

Elimina cartella condivisa

Elimina la cartella condivisa selezionata.

ATTENZIONE:l'eliminazione della cartella condivisa comporta l'eliminazione di tutti i dati in essa contenuti.
